Sorted Doubly Linked List

Welcome to my implementation of a sorted DLL! I made this project to help me learn the basics of JavaScript and HTML. Navigating through this web page allows you to perform all of the basic functions on a sorted doubly linked list, which I have implemented in JavaScript.

All code was written by Shawn O'Grady

Some important notes about this program:

  1. Each value in the list needs to be a number
    • Trying to add a non-number value like hello or one will trigger a pop-up
  2. Each value may appear in the list up to twice
    • i.e. each value may only have one duplicate
    • trying to add the same value a third time will trigger a pop-up message
